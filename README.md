### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/TusharKaundal/unada_labs_assignment.git
   cd unada_labs_assignment
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn
   ```

3. **Start the development server**

   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open in browser**
   The application will be available at `http://localhost:5173`
